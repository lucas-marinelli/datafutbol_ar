# Template — Thread X / Twitter

**Estructura:** 3-6 tweets máximo. Cada tweet con 1 imagen propia.

**Tweet 1 (hook):**
- 1ra línea = afirmación fuerte o pregunta
- 2da línea = qué viene en el thread
- (1/N) o 🧵👇 al final
- 1 imagen estrella

**Tweets 2 a N-1 (desarrollo):**
- (n/N) al inicio
- 1 idea + 1 imagen por tweet
- Sin hashtags acá

**Último tweet (cierre):**
- Conclusión + CTA (DM, follow, comentar, link a IG)
- 1-2 hashtags si querés

---

## Ejemplo (Episodio 2 thread X 4 tweets)

**Tweet 1 (con shot map):**
```
5 visualizaciones para entender el fútbol con datos.

(1/4) SHOT MAP — Final del Mundial 2022.
Cada círculo = un remate. Tamaño = xG. Color lleno = gol.

A simple vista: quién tiró, desde dónde, qué tan probable era el gol.

🧵👇
```

**Tweet 2 (con radar):**
```
(2/4) RADAR DE PERCENTILES — Messi 14/15 vs delanteros top de Europa.

Cada rebanada = una métrica. Más larga = mejor que el resto.

Útil para scouting: te dice si un jugador es "top 10%" o "top 50%" en algo.
```

**Tweet 3 (con pass network):**
```
(3/4) PASS NETWORK — Argentina 1er tiempo vs Francia.

Nodos = jugadores. Líneas = pases entre dos.
A simple vista vés: quién tocó más la pelota y la forma del equipo.
```

**Tweet 4 (con heatmap + CTA):**
```
(4/4) HEATMAP — Messi en la final del Mundial 2022.

Densidad de acciones por zona. Cuanto más claro, más tocó.

Estas 4 + radar = base de todo lo que vas a ver acá.

¿Cuál te explico primero? 👇
```

## Tips de imágenes para X

- Resolución: 1600×900 (paisaje) o 1080×1080 (cuadrada). Evitá 1080×1350 acá (queda cortada).
- En el feed, X muestra ~16:9. Asegurate que el título quede arriba y el watermark abajo.
