# outputs/

Artefactos finales (PNG/PDF) listos para publicar. Una subcarpeta por mes.

## Convención de nombrado

```
YYYY-MM-DD_TITULO_FORMATO.png
```

Ejemplos:
- `2026-05-11_episodio2_shotmap_arg_fra.png`
- `2026-05-11_episodio2_radar_messi.png`
- `2026-05-15_xg_explicado_carrusel_slide3.png`
- `2026-06-01_lista26_radar_central_zurdo.pdf`

## Tipos de archivo

| Formato | Para qué | Tamaño típico |
|---|---|---|
| `.png` IG carrusel | Post Instagram carrusel | 1080×1350 |
| `.png` IG cuadrado | Post Instagram feed | 1080×1080 |
| `.png` IG Reel cover | Portada de Reel | 1080×1920 |
| `.png` X imagen | Post X / Twitter | 1600×900 o 1080×1080 |
| `.pdf` informe | Scouting, reportes largos | A4 |

## Lo que NO va acá

- Borradores (`draft_*`, `test_*`, `_tmp_*`) — están en .gitignore
- Datos crudos — esos van a `data/`
- Plantillas reutilizables — esas van a `templates/`
