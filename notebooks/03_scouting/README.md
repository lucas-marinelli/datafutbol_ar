# 03 — Scouting

Notebooks de informes de scouting (radares, comparables, perfiles completos). Estos son la base de los **informes vendibles** que arrancan en mes 4-6.

Convención:
```
NN_AAAA-MM-DD_jugador_apellido.ipynb
```
Ejemplo: `01_2026-05-15_mastantuono_franco.ipynb`

**Estructura mínima del informe (10 secciones — ver `00_Plan/` para el detalle):**
1. Datos contractuales y físicos
2. Resumen ejecutivo
3. Perfil técnico-táctico (3 fortalezas + 3 debilidades)
4. Métricas avanzadas en radar percentil
5. Shot map + heat map + pass network
6. 3 comparables
7. Proyección
8. Riesgos
9. Recomendación + rango de mercado
10. Output: PDF 8-12 páginas en `outputs/YYYY-MM/scouting_apellido.pdf`
