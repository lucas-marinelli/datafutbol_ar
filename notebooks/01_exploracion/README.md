
# 01 — Exploración

Notebooks de análisis exploratorio: cargar datasets nuevos, jugar con métricas, validar hipótesis antes de armar un post.

**No tienen que terminar en publicación.** Acá probás libremente.

Convención de nombrado:
```
NN_AAAA-MM-DD_tema_corto.ipynb
```
Ejemplo: `01_2026-05-12_xg_liga_argentina.ipynb`

Si un notebook acá se convierte en post, copialo a `02_posts/` con su título final.
